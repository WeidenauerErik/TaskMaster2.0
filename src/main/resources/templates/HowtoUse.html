About our Project

The TaskMaster project is a web application that is used to manage the homework and assignments of a class in a clear and centralized way. Teachers can log in with their login data and assign room-tasks. The students on the other hand can view these tasks under their own access.
Our application is based on SpringBoot using Thymeleaf. SpringBoot to create a web app with Java and manage the project. With Thymeleaf DOM elements for the HTML code can be created, which is important for creating new tasks.

The goal of the project is to learn how to manage the login of users and the storage of data, as well as to become more familiar with SpringBoot and Thymeleaf.
